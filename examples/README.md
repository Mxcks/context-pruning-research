# Context Pruning Implementation Examples

This directory contains example implementations of the context pruning system.

## Examples

1. **basic_pruning.py** - Simple implementation of context pruning with node-based isolation
2. **tagging_system.py** - Example of the tagging-based selection mechanism
3. **lifecycle_demo.py** - Demonstration of the context lifecycle (active → compressed → detached)

## Usage

Each example can be run independently to demonstrate different aspects of the context pruning system:

```bash
# Run basic pruning example
python examples/basic_pruning.py

# Run tagging system example
python examples/tagging_system.py

# Run lifecycle demo
python examples/lifecycle_demo.py
```
